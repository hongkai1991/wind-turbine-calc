## 计算流程

1. **基础体型验算** - 验证基础几何尺寸是否合理
2. **基础自重计算** - 计算基础自重、回填土重量和浮力等
3. **塔筒底部风机荷载计算** - 基于风机信息和塔筒段信息
4. **四种工况荷载处理** - 
5. **荷载组合计算** - 包括：
   - 荷载标准值计算
   - 基础总恒载计算
   - 水平力矩之和计算
   - 偏心距计算
   - 基础底面净反力计算
   - 基础底面最大/最小净反力计算
   - 基本组合设计值计算
   - 基本组合压力计算

3. 四种工况的参数将按照对应的 `LoadCase` 枚举值进行处理：
   - `normal` → `LoadCase.NORMAL`
   - `extreme` → `LoadCase.EXTREME`
   - `fatigueUpper` → `LoadCase.FATIGUE_UPPER`
   - `fatigueLower` → `LoadCase.FATIGUE_LOWER`
