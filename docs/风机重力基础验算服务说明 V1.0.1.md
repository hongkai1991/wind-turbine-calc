# 风机重力基础验算服务使用说明
V1.0.1 

## 概述
本服务提供风机重力基础的综合验算功能，包括基础设计验证、承载力分析、沉降计算、抗冲切验算、抗倾覆验算、刚度验算等完整的基础设计验算流程。

## 版本更新说明
- V1.0.1: 增加了基础参数、设计参数、允许脱空面积和刚度要求等新功能模块
- 优化了接口结构，提供更完善的基础设计验算能力

## 计算流程

V1.0.1 版本提供完整的风机重力基础验算流程：

1. **基础体型验算** - 验证基础几何尺寸是否合理
2. **基础自重计算** - 计算基础自重、回填土重量和浮力等
3. **塔筒底部风机荷载计算** - 基于风机信息和塔筒段信息
4. **六种工况荷载处理** - 正常工况、极端工况、疲劳工况上限、疲劳工况下限、多遇地震工况、罕遇地震工况
5. **荷载组合计算** - 包括：
   - 荷载标准值计算
   - 基础总恒载计算
   - 水平力矩之和计算
   - 偏心距计算
   - **偏心距安全检查**（e/r1 > 0.52 时提供设计不合理警告）
   - 基础底面净反力计算
   - 基础底面最大/最小净反力计算
   - 基本组合设计值计算
   - 基本组合压力计算
6. **地基承载力验算** - 验证地基承载能力是否满足要求
7. **沉降分析** - 计算基础沉降量
8. **抗倾覆验算** - 验证基础抗倾覆稳定性
9. **脱空面积验算** - 检查基础脱空面积是否在允许范围内
10. **抗冲切验算** - 验证基础抗冲切承载力
11. **刚度验算** - 验证基础旋转刚度和水平刚度是否满足要求

## V1.0.1 响应结果说明

响应数据结构包含以下主要模块：

### 基础验算模块
- `geometry_validation`: 基础体型验算结果
- `self_weight`: 基础自重计算结果  
- `tower_base_load`: 塔筒底部荷载计算结果（含地震力分析）
- `load_analysis`: 荷载分析结果（包含六种工况的详细计算）

### 新增验算模块（V1.0.1）
- `detachment_area_analysis`: 脱空面积验算结果
- `bearing_capacity_analysis`: 地基承载力验算结果
- `anti_overturning_analysis`: 抗倾覆验算结果
- `anti_sliding_analysis`: 抗滑移验算结果
- `stiffness_analysis`: 刚度验算结果
- `shear_strength_analysis`: 抗剪强度验算结果  
- `anti_punching_analysis`: 抗冲切验算结果

### 设计参数与汇总
- `design_parameters`: 设计参数汇总
- `summary`: 计算汇总信息

### 错误处理增强
- 当 e/r1 > 0.52 时，`load_analysis.summary` 将包含：
  - `design_error_cases`: 设计错误工况数量
  - `design_warnings`: 具体设计警告信息
  - `design_recommendations`: 设计改进建议

## 注意事项

### 参数要求
1. **必填参数**：所有参数组（baseParameters、designParameters、geometry、material、soilLayers、windTurbine、towerDrums、turbineLoads）均为必填项
2. **单位说明**：
   - 力：kN
   - 弯矩：kN·m  
   - 长度：m
   - 应力：N/mm² 或 kPa
   - 密度/容重：kN/m³
   - 角度：度（°）
   - 刚度：N·m/rad（旋转刚度）、N/m（水平刚度）

### 工况处理
3. **六种工况**的参数将按照对应的 `LoadCase` 枚举值进行处理：
   - `normal` → `LoadCase.NORMAL` （正常工况）
   - `extreme` → `LoadCase.EXTREME` （极端工况）
   - `fatigueUpper` → `LoadCase.FATIGUE_UPPER` （疲劳工况上限）
   - `fatigueLower` → `LoadCase.FATIGUE_LOWER` （疲劳工况下限）
   - 多遇地震工况和罕遇地震工况将基于正常工况和地震力自动计算

### 安全检查
4. **偏心距安全检查**：当偏心距与基础半径比值 e/r1 > 0.52 时，系统将：
   - 抛出基础设计不合理警告
   - 提供详细的错误信息和改进建议
   - 建议增大基础尺寸或减小作用荷载

### 新增功能
5. **脱空面积控制**：可通过 `allowedDetachmentArea` 设置不同工况下的允许脱空面积比例
6. **刚度验算**：可通过 `stiffnessRequirements` 设置基础刚度要求
7. **设计参数优化**：支持不同安全等级和重要性系数的设计

### 兼容性
8. **向后兼容**：接口保持向后兼容，但建议使用新的参数结构以获得完整功能
